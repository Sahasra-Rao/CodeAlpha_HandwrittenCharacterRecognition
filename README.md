# Handwritten Character Recognition using CNN

## Objective
Build a machine learning model to recognize handwritten digits using the MNIST dataset.

## Dataset
MNIST Handwritten Digits Dataset

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Scikit-learn

## Deep Learning Model
- Convolutional Neural Network (CNN)

## Project Workflow
1. Load MNIST Dataset
2. Data Preprocessing
3. CNN Model Creation
4. Model Training
5. Model Evaluation
6. Prediction of Handwritten Digits

## Evaluation Metrics
- Accuracy
- Classification Report
- Confusion Matrix

## Conclusion
The CNN model successfully recognizes handwritten digits with high accuracy, demonstrating the effectiveness of deep learning for image classification tasks.
